# TextEncoder


TextDecoder 接口表示一个文本解码器，一个解码器只支持一种特定文本编码，例如 UTF-8、GB2312、GBK，等等。解码器将字节流作为输入，并提供码位流作为输出。

当解码器在解码时遇到格式错误的数据时，解码器将使用替换字符 U+FFFD（�）替换无效的数据。

[MDN Reference](https://developer.mozilla.org/docs/Web/API/TextDecoder)
