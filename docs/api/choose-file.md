## tips
- chooseFile是系统UI，其风格不同rom可能有差异。多选时有的是长按、有的是checkbox。系统UI的暗黑模式、国际化跟随系统，而不跟随App。
- chooseFile不仅仅是选择现有文件，有的手机也可以新拍照、拍摄、录音。但不同系统的实现存在差异。
